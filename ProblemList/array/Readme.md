#### ☁︎ 배열의 삭제

```javascript

다음 배열에서 400, 500를 삭제하는 code를 입력하세요.

var nums = [100, 200, 300, 400, 500];

```

<details><summary><b>Answer</b></summary>
  <p>

```javascript
var nums = [100, 200, 300, 400, 500];
nums.splice(3, 2);
console.log(nums);
```

 </p>
