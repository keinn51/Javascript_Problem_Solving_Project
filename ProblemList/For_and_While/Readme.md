#### ☁︎ for문 계산


 ```javascript

다음 코드의 출력 값으로 알맞은 것은?


var a = 10;
var b = 2;
for(var i=1; i<5; i+=2){
 a += i;
}
console.log(a+b);

````



<details><summary><b>Answer</b></summary>

  <p>

```javascript
16

=> a에 1과 3이 더해지고, b=2 이므로 10 + 1 + 3 + 2 = 16
````
